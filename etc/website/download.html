Java JRE 1.7 (Java 7) must be installed in order to run MOST. Java can be installed at java.com/download.

How to determine if Java is installed and what version:

Windows 7 - Control Panel:

Change the View Category to either Large Icons or Small Icons and locate the Java icon. (If there is no Java icon, Java is not installed.) Click on the icon or the word "Java" and then click on the About button. If the About Java box says "Version 7", the correct version of Java is installed.

Windows XP - Control Panel:

Same as above but no need to change the View.

Windows 7 or XP - Command Line:

Open a command window and type:
java -version
The output will look something like this:
java version "1.7.0_45"
If the version starts with 1.7 the correct version of Java is installed.

Linux:

Open a terminal and type:
java -version
The output will look something like this:
java version "1.7.0_45"
If the version starts with 1.7 the correct version of Java is installed.
